关于链表

定义

链表是一种将数据存储到“结点”中的数据结构，需要存储多少个数据，就生成多少个“结点”，把这些“结点”用指针挂接起来。
为了表示每个数据元素ai与其直接后继元素ai+1之间的逻辑关系，对数据元素ai来说，除了存储其本身的信息之外，还需存储一个指示其直接后继的信息(即直接后继的存储位置)，这两部分信息组成数据元素ai的存储映像，称为结点。
结点中包括两个域，其中存储数据元素信息的域称为数据域；存储直接后继存储位置的域称为指针域。指针域中存储的信息称为指针。

这些是书上的概念

简单来说就是，定义一个结构体

struct LinkNode
{
    int data; //此处为数据域
    struct LinkNode *next //此处为指针域
};

一个链表由头，结点和尾组成

其实这个头就是指向第一个结点的指针，最后的尾则是最后一个结点的指向的下一个地址为 NULL

其实就是这玩意

这个是单向链表，除此之外常见的还有

  • 双向链表
    • 每个结点中保存了上一个结点的地址*prev
  • 单向循环链表
    • 最后一个结点的*next指向头
  • 双向循环链表
    • 最后一个结点的*next指向头，每个结点中保存了上一个结点的地址*prev

对链表的操作

插入新结点

利用malloc()动态申请内存作为新结点插入到链表中

例如：

struct ListNode *add;
add = (struct ListNode *)malloc(sizeof(struct ListNode));

可以生成一个新结点，在生成后可对该新节点的数据域进行赋值操作，并插入链表中

一般来说，对单链表插入新结点有三种方法

代码示例

例题：列表拼接（来源：PTA|程序设计类实验辅助教学平台）

本题要求实现一个合并两个有序链表的简单函数。链表结点定义如下：

struct ListNode {
    int data;
    struct ListNode *next;
};

函数接口定义：

struct ListNode *mergelists(struct ListNode *list1, struct ListNode *list2);

其中list1和list2是用户传入的两个按data升序链接的链表的头指针；函数mergelists将两个链表合并成一个按data升序链接的链表，并返回结果链表的头指针。

裁判测试程序样例：

#include <stdio.h>
#include <stdlib.h>

struct ListNode {
    int data;
    struct ListNode *next;
};

struct ListNode *createlist(); /*裁判实现，细节不表*/
struct ListNode *mergelists(struct ListNode *list1, struct ListNode *list2);
void printlist( struct ListNode *head )
{
     struct ListNode *p = head;
     while (p) {
           printf("%d ", p->data);
           p = p->next;
     }
     printf("\n");
}

int main()
{
    struct ListNode  *list1, *list2;

    list1 = createlist();
    list2 = createlist();
    list1 = mergelists(list1, list2);
    printlist(list1);

    return 0;
}

1 3 5 7 -1

struct ListNode *mergelists(struct ListNode *list1, struct ListNode *list2)
{
    if (list1 == NULL)
    {
        return list2;
    }
    if (list2 == NULL)
    {
        return list1;
    }

    struct ListNode *checkpoint, *add, *last;

    while (list2 != NULL)
    {
        add = list2;
        list2 = list2->next;
        if (add->data < list1->data)
        {
            add->next = list1;
            list1 = add;
            continue;
        }

        checkpoint = list1;
        while(checkpoint!=NULL)
        {
            if(checkpoint->next==NULL)
            {
                checkpoint->next = add;
                add->next = NULL;
                break;
            }
            if(checkpoint->next->data>add->data)
            {
                add->next=checkpoint->next;
                checkpoint->next = add;
                break;
            }
            checkpoint=checkpoint->next;
        }
    }
    return list1;
}
